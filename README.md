# JavaScript Fundamentals Tutorial

A beginner-friendly repository designed to explore the core concepts of JavaScript. This project is structured as a series of interactive HTML/JS modules, allowing you to see exactly how different data types and declarations behave in the browser.

## 🚀 Getting Started

To interact with this tutorial:
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser to access the main Learning Hub.
3. Click on any of the module links to load a specific topic.
4. **Crucial Step:** Open your browser's Developer Tools (usually by pressing `F12`, `Ctrl+Shift+I`, or `Cmd+Opt+I`) and navigate to the **Console** tab. The visual output happens here!

## 📂 Repository Structure & Modules

This learning hub is broken down into the following key concepts:

### 1. Basics & Console (`index.js`)
* Covers basic dynamic typing.
* Explains the nuances of using `console.log()`, specifically the difference between using a comma `,` (multi-argument separation) and a plus `+` (string concatenation).

### 2. Variables (`variables.html` & `variables.js`)
* Explains variable declaration using `let`.
* Demonstrates runtime type assignment.
* Outlines strict naming conventions and rules (e.g., case sensitivity, no reserved keywords).

### 3. Constants (`constants.html` & `constants.js`)
* Demonstrates the use of `const`.
* Shows the deliberate error thrown when attempting to reassign a constant value, reinforcing immutability concepts.

### 4. Primitive Data Types (`primitives.html` & `primitiveDataTypes.js`)
Explores the basic building blocks of data:
* `String` literals
* `Number` literals
* `Boolean` literals
* `Undefined`
* `Null`

### 5. Reference Data Types (`reference.html` & `referenceDataTypes.js`)
Explores more complex data structures:
* **Objects:** Creating object literals `{}` and accessing properties using both dot notation (`person.name`) and bracket notation (`person['name']`).
* **Arrays:** Creating array literals `[]`, accessing elements by index, and understanding the `.length` property.
* **Functions:** Declaring functions, passing parameters for interactivity, and using the `return` keyword.
